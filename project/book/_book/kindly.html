2.4.11 Plain data structures

Plain Clojure data structures have recursive kind semantics: - Each tool has its own way to represent them visually (e.g., Clay just uses text, while Portal has a hierarchical navigation UI). - If the values inside them have kind information, they are handled accordingly.

Examples:

(list 1 "A" :B 'C)
(1 "A" :B C)
(clay/in-portal
 (list 1 "A" :B 'C))
[1 "A" :B 'C]
[1 "A" :B C]
(clay/in-portal
 [1 "A" :B 'C])
#{1 "A" :B 'C}
#{1 :B C "A"}
(clay/in-portal
 #{1 "A" :B 'C})
{1 "A" :B 'C}
{1 "A", :B C}
(clay/in-portal
 {1 "A" :B 'C})
